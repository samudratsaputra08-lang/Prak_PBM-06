# CRUD Mahasiswa Flutter SQLite

Aplikasi CRUD (Create, Read, Update, Delete) data mahasiswa menggunakan Flutter dan SQLite.

## Pembuat

## Fitur
- Tambah data mahasiswa
- Menampilkan data mahasiswa
- Edit data mahasiswa
- Hapus data mahasiswa
- Database SQLite

## Teknologi
- Flutter
- Dart
- SQLite (sqflite)
